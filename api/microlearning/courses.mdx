## Create Course

Create a new microlearning course with multimedia content and quizzes.

### Endpoint

```
POST /api/microlearning/courses
```

### Authentication

Requires authentication and **admin** role.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

### Request Body

<ParamField body="title" type="string" required>
  Course title. Must be between 5-200 characters.
</ParamField>

<ParamField body="description" type="string" required>
  Course description. Must be between 20-1000 characters.
</ParamField>

<ParamField body="area" type="string" required>
  Target area for the course. Options: `enfermeria`, `urgencias`, `maternidad`, `administracion`, `general`
</ParamField>

<ParamField body="content" type="object" required>
  Course multimedia content
  
  <Expandable title="content properties">
    <ParamField body="content.videoUrl" type="string">
      URL to course video content
    </ParamField>
    
    <ParamField body="content.audioUrl" type="string">
      URL to podcast/audio content
    </ParamField>
    
    <ParamField body="content.pdfUrl" type="string">
      URL to infographic/PDF content
    </ParamField>
    
    <ParamField body="content.duration" type="number" required>
      Course duration in minutes (1-120)
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="quiz" type="array" required>
  Array of quiz questions (minimum 5, maximum 15 questions)
  
  <Expandable title="quiz item properties">
    <ParamField body="quiz[].question" type="string" required>
      Question text (minimum 10 characters)
    </ParamField>
    
    <ParamField body="quiz[].options" type="array" required>
      Array of answer options (2-5 options)
    </ParamField>
    
    <ParamField body="quiz[].correctAnswer" type="number" required>
      Index of the correct answer (0-based)
    </ParamField>
    
    <ParamField body="quiz[].points" type="number">
      Points awarded for correct answer. Default: 1
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="passingScore" type="number">
  Minimum score to pass (50-100). Default: 70
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the course was created successfully
</ResponseField>

<ResponseField name="message" type="string">
  Success message
</ResponseField>

<ResponseField name="data.course" type="object">
  The created course object with all properties and MongoDB `_id`
</ResponseField>

## Get Available Courses

Retrieve a paginated list of active courses with optional filtering.

### Endpoint

```
GET /api/microlearning/courses
```

### Authentication

Requires authentication.

### Query Parameters

<ParamField query="area" type="string">
  Filter courses by area: `enfermeria`, `urgencias`, `maternidad`, `administracion`, `general`
</ParamField>

<ParamField query="page" type="number">
  Page number for pagination. Default: 1
</ParamField>

<ParamField query="limit" type="number">
  Number of courses per page. Default: 10
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="data.courses" type="array">
  Array of course objects (only active courses)
  
  <Expandable title="course properties">
    <ResponseField name="_id" type="string">
      Course unique identifier
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Course title
    </ResponseField>
    
    <ResponseField name="description" type="string">
      Course description
    </ResponseField>
    
    <ResponseField name="area" type="string">
      Course area
    </ResponseField>
    
    <ResponseField name="content.duration" type="number">
      Course duration in minutes
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="data.pagination" type="object">
  Pagination metadata
  
  <Expandable title="pagination properties">
    <ResponseField name="page" type="number">
      Current page number
    </ResponseField>
    
    <ResponseField name="limit" type="number">
      Items per page
    </ResponseField>
    
    <ResponseField name="total" type="number">
      Total number of courses
    </ResponseField>
    
    <ResponseField name="pages" type="number">
      Total number of pages
    </ResponseField>
  </Expandable>
</ResponseField>

### Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.cuido.com/api/microlearning/courses?area=urgencias&page=1&limit=10" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.cuido.com/api/microlearning/courses?area=urgencias&page=1&limit=10', {
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});

const data = await response.json();
```
</CodeGroup>
