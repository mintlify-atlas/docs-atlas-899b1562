---
title: 'Update Employee'
api: 'PUT /api/employees/:id'
description: 'Update employee information including wellness metrics and gamification data'
---

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<Note>
  This endpoint requires **admin** privileges.
</Note>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique MongoDB identifier of the employee to update
</ParamField>

## Request Body

All fields are optional. Only include the fields you want to update.

<ParamField body="personalInfo" type="object">
  Update personal information
  
  <Expandable title="personalInfo properties">
    <ParamField body="personalInfo.name" type="string">
      Full name (max 100 characters)
    </ParamField>
    
    <ParamField body="personalInfo.email" type="string">
      Email address (must be unique)
    </ParamField>
    
    <ParamField body="personalInfo.identification" type="string">
      ID number (must be unique)
    </ParamField>
    
    <ParamField body="personalInfo.phoneNumber" type="string">
      Phone number
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="jobInfo" type="object">
  Update employment information

    <ParamField body="jobInfo.position" type="string">

    <ParamField body="jobInfo.department" type="string">

<ParamField body="hospitalId" type="string">

    <ParamField body="wellnessMetrics.teamSupportScore" type="number">

## Common Update Scenarios

### Update Wellness Assessment

After a wellness check-in, update the employee's metrics:

```javascript
await updateEmployee(employeeId, {
  wellnessMetrics: {
    currentMoodScore: 4,
    averageWorkload: 5,
    teamSupportScore: 4,
    satisfactionScore: 7,
    riskLevel: 'medio'
  }
});
```

### Award Points and Badge

When an employee completes an activity:

```javascript
const employee = await getEmployee(employeeId);
const newPoints = employee.gamification.totalPoints + 50;
const newLevel = Math.floor(newPoints / 100) + 1;

await updateEmployee(employeeId, {
  gamification: {
    totalPoints: newPoints,
    level: newLevel,
    badges: [
      ...employee.gamification.badges,
      {
        name: 'Actividad Completada',
        earnedAt: new Date().toISOString(),
        description: 'Completó un ejercicio de mindfulness'
      }
    ]
  }
});
```

### Change Department

When an employee transfers to a new department:

```javascript
await updateEmployee(employeeId, {
  jobInfo: {
    department: 'hospitalizacion',
    shift: 'mañana'
  }
});
```

### Deactivate Employee

When an employee leaves:

```javascript
await updateEmployee(employeeId, {
  isActive: false
});
```
